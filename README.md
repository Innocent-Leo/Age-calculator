# Age Calculator

A simple web application that calculates a user's age based on their birth date, demonstrating input validation and date-based calculations.

---

## Overview

This project focuses on implementing logic for calculating age while handling user input and validating date values.

It demonstrates practical JavaScript logic combined with structured user interface feedback.

---

## Built With

- HTML5
- CSS3
- JavaScript

---

## Features

- Age calculation based on date input
- Input validation and error handling
- Responsive user interface
- Dynamic result updates
